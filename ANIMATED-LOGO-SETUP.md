# الشعار المتحرك — صيدلية سما السكر

مكوّن شعار متحرك فاخر مبني على GSAP، يستخدم **نفس الشعار المعتمد حرفياً**
(تم استخراج المسارات الرقمية من `public/logo.png` بتتبّع نقطي دقيق —
بدون أي إعادة تصميم أو تغيير في الألوان أو النسب).

## الملفات

| الملف | الوصف |
|---|---|
| `src/components/logo/AnimatedLogo.tsx` | المكوّن الجاهز (client component) |
| `src/components/logo/animation.ts` | مصنع خطوط GSAP الزمنية مع شرح كل حركة |
| `src/components/logo/logo-paths.ts` | بيانات المسارات المستخرجة من الشعار الأصلي |
| `logo-animation-preview.html` | معاينة فورية — افتحه في المتصفح مباشرة |

## التثبيت

```bash
npm install gsap
```

هذا كل شيء — لا حاجة لأي إعداد آخر.

## الاستخدام

```tsx
import AnimatedLogo from "@/components/logo/AnimatedLogo";

// في الهيدر
<AnimatedLogo className="w-12 h-12" />

// في صفحة البداية / شاشة تحميل
<AnimatedLogo className="w-40 md:w-56" />
```

## تسلسل الحركة (6 ثوانٍ)

1. **0.0 – 1.6s** — إطار الكبسولة يُرسم بانسيابية (stroke draw-on)
2. **1.15 – 2.15s** — الشعار الكامل يظهر بتلاشٍ هادئ
3. **حلقة مستمرة (6s):**
   - نسيم خفيف في الشعر فقط (إزاحة ≤ 2.6px — الوجه والإطار ثابتان)
   - وميض لافندر ينساب عبر الشعر
   - النجمة تتلألأ مرة واحدة (شفافية فقط — بلا تكبير أو دوران)
   - مسحة ضوء فاخرة تعبر الشعار
   - طفو عمودي ±3px بإيقاع 6 ثوانٍ

## الأداء وإمكانية الوصول

- كل الحركات opacity/transform (مسرّعة بكرت الشاشة، 60fps)
- فلتر الإزاحة مقصوص على منطقة الشعر فقط لتقليل الحساب
- `prefers-reduced-motion` → شعار ثابت بلا أي حركة
- معرّفات فريدة (useId) — يمكن وضع أكثر من شعار في نفس الصفحة

## الوضع الداكن

ألوان العلامة تبقى كما هي (الكبسولة الكريمية معتمة فتظهر ممتازة على
الخلفيات الداكنة). لتوهّج ناعم اختياري في الداكن:

```tsx
<AnimatedLogo className="w-32 dark:drop-shadow-[0_0_24px_rgba(124,79,136,0.35)]" />
```

## استبدال شعار الهيدر الحالي (اختياري)

في `src/components/layout/header.tsx` استبدل `<img src="/logo.png" ...>` بـ:

```tsx
<AnimatedLogo className="w-full h-full" />
```
